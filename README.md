## 💁‍♀️ 개발 담당 부분
* **조현진 - 메인화면 다이얼로그, 플레이리스트뷰**
  * 퍼즈 시작하기 클릭 시 다이얼로그 - 시작하기 클릭 - 플레이리스트뷰로 이동
  * 플레이리스트뷰에서 아이템 클릭 시 유튜브 재생목록 url로 이동, timer 시작
  
* **박효송 - 스플래시, 시청기록뷰**
  * 스플래시뷰 1초 구동
  * 시청기록뷰에서 막대그래프로 일주일치 시청기록 보여줌
  
* **권예진 - 로그인뷰, 메인뷰, 종료뷰**
  * 로그인 버튼 클릭 시 메인뷰로 이동
  * 하단 버튼 클릭 시 시청기록뷰로 이동
